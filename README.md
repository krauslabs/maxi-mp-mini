# maxi-mp-mini
A Customized Set of Mods and Parts for the Monoprice Select Mini (v2)

## Modifications

### 40mm Hotend Fan Shroud

The 30mm hotend and part cooling fan on the stock MPSM is quite noisy. Replacing it with a Noctua 40x20mm fan greatly reduces noise and improves cooling. This shroud is a redesign in the style of [a similar shroud by keco185 on thingiverse](https://www.thingiverse.com/thing:2448223).

#### Required Components
- Any 40mm 12V Fan (Noctua NF-A4x10 FLX recommended)
- 40mm-Fan-Shroud.stl

### TODO: Linear Rod Replacements

This mod replaces all the linear rods with higher quality and larger diameter rods, to improve kinematic accuracy and print quality. At the same time, the belts are upgraded to GT2 belts, and all bearings and idlers are replaced with metal versions.
